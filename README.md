# 企业财务账目录入与利润核算系统

## 项目介绍

企业财务账目录入与利润核算系统是一款专为中小企业设计的财务管理软件，提供全面的财务数据录入、管理、分析和报表生成功能。该系统采用现代化的分层架构设计，具有操作简便、功能完善、数据安全等特点。

## 系统功能

### 1. 基础数据管理
- 账户管理：创建、编辑、删除各种类型的账户（银行账户、现金账户等）
- 分类管理：建立多级收支分类体系，支持分类树状结构展示
- 用户管理：多用户支持，基于角色的权限控制

### 2. 财务录入功能
- 交易记录录入：支持收入、支出、转账等多种交易类型
- 批量导入：支持Excel批量导入交易数据
- 智能填充：自动补全账户、分类等信息，提高录入效率

### 3. 利润核算分析
- 实时利润计算：自动汇总收入支出，计算利润
- 成本分析：按分类、项目、时间等维度分析成本构成
- 趋势分析：提供利润走势图表，支持多维度对比

### 4. 报表生成
- 利润表：自动生成月度、季度、年度利润报表
- 资产负债表：反映企业财务状况
- 现金流量表：追踪资金流动情况
- 自定义报表：支持用户自定义报表内容和格式

### 5. 数据可视化
- 仪表盘：直观展示关键财务指标
- 收支对比图：可视化收入支出对比情况
- 分类饼图：展示各分类占比
- 趋势折线图：展示财务数据随时间变化趋势

### 6. 系统设置
- 主题切换：支持明暗两种主题
- 数据备份与恢复：定期自动备份，支持手动备份恢复
- 系统参数配置：灵活配置系统各项参数

## 技术架构

### 系统架构
- **前端**：PyQt5图形用户界面，支持Windows平台
- **后端**：Python业务逻辑处理
- **数据库**：SQLite轻量级数据库
- **数据可视化**：Matplotlib图表生成

### 核心技术栈
- **Python 3.8+**：主要开发语言
- **PyQt5**：桌面GUI框架
- **SQLite**：本地数据库
- **Pandas**：数据处理
- **Matplotlib**：数据可视化
- **Passlib**：密码加密

## 安装指南

### 环境要求
- Windows 7/8/10/11 64位操作系统
- Python 3.8 - 3.10
- 至少2GB内存
- 至少100MB可用磁盘空间

### 使用安装包安装
1. 下载最新版本的安装包（finance-management-system-setup.exe）
2. 双击运行安装程序
3. 按照安装向导的提示完成安装
4. 安装完成后，可以从桌面快捷方式或开始菜单启动系统

### 从源码安装
1. 确保已安装Python 3.8或更高版本
2. 克隆或下载项目源码
3. 安装依赖项
   ```
   pip install -r requirements.txt
   ```
4. 运行应用程序
   ```
   python main.py
   ```

## 配置与部署

### 数据库配置
系统默认使用SQLite数据库，数据文件存储在应用程序目录的`data`文件夹中。无需额外的数据库配置。

### 环境变量配置
可选的环境变量配置：
- `FINANCE_DATA_DIR`：自定义数据存储目录
- `FINANCE_LOG_LEVEL`：日志级别（DEBUG, INFO, WARNING, ERROR）

### 多用户部署
1. 在用户管理界面创建多个用户账户
2. 为不同用户分配适当的角色和权限
3. 各用户使用自己的账户登录系统

## 数据安全

### 数据备份
- 系统支持自动和手动备份
- 自动备份默认每周执行一次
- 建议定期将备份文件复制到外部存储设备

### 数据恢复
- 通过系统设置中的恢复功能恢复数据
- 支持从指定备份文件恢复

### 密码安全
- 用户密码采用加盐哈希存储
- 支持定期密码更新策略
- 登录失败次数限制，防止暴力破解

## 系统维护

### 日志管理
- 系统日志存储在`logs`目录
- 定期清理旧日志文件，释放磁盘空间

### 性能优化
- 定期运行数据库优化功能
- 对于大量数据，考虑定期归档历史数据

### 常见问题排查
1. **登录失败**：检查用户名密码是否正确，账户是否被锁定
2. **数据显示异常**：尝试刷新页面或重启应用
3. **报表生成失败**：检查数据完整性，确保日期范围正确
4. **系统响应缓慢**：关闭其他占用资源的应用，或清理数据库
